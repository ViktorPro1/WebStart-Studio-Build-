## 🏗️ v1.9.0 - Build-процес CSS (31.10.2025)

### 🎯 Основні Зміни

**Автоматизація Білду**

- Створено `concat-css.js`
- NPM скрипти для білду
- Генерація `dist/styles.css`

### 🛠️ Використання

```bash

# Автоматичний білд при змінах
npm run watch:css
```

### 📂 Результат

**Розробка:**

```html

```

**Продакшн:**

```html
<link rel="stylesheet" href="dist/styles.css" />
```

## 📚 Changelog Format

Ми дотримуємося [Keep a Changelog](https://keepachangelog.com/uk/1.0.0/) та [Semantic Versioning](https://semver.org/lang/uk/).

### Типи Змін

- `Added` - Нові функції
- `Changed` - Зміни існуючого функціоналу
- `Deprecated` - Функції, що будуть видалені
- `Removed` - Видалені функції
- `Fixed` - Виправлення багів
- `Security` - Виправлення вразливостей
